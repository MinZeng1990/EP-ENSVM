# EP-ENSVM
# Description
EP-ENSVM is a computational approach for essential protein prediction by using only sequence information. 

# Requirements
numpy==1.18.1

scikit-learn==0.23.1

imblearn==0.7.0

python train_main.py --type h --threshold 0.5
